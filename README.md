# RiftSense

Undergraduate Computer Science Research Project - UFABC

RiftSense is a research project focused on detecting implicit and explicit conflicts in social media interactions using Transformer-based language models.

The project includes:

- dataset construction
- interaction pair generation
- manual annotation
- inter-annotator agreement analysis
- BERTimbau fine-tuning
- performance evaluation

## Objective

The goal of RiftSense is to investigate computational approaches for identifying antagonistic interactions in Brazilian Portuguese social media discourse, including implicit manifestations such as sarcasm, provocation and criticism, as well as explicit hostility.

## Dataset
The dataset is composed of Brazilian Portuguese Twitter interactions manually annotated according to the presence or absence of conflict in pairs of texts.

## Pipeline

1. Raw tweet collection
2. Dataset construction
3. Interaction pair generation
4. Manual annotation
5. Agreement analysis
6. Model training
7. Evaluation
